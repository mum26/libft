# Usage

Add "libft.a" when compiling the "~.c" file.

```bash
$ cc your/file/path/~.c your/libft/path/libft.a
```

# Installation

```bash
$ git clone https://github.com/mum26/libft.git
$ cd libft
```

# Compile

```bash
$ make
```

### With_bonus

```bash
$ make bonus
```
